# svelte-vite-ssr

### Run in SSR mode (dev)
```sh
npm run dev:ssr
```

### Run in CSR mode (dev)
```sh
npm run dev:csr
```

### Build SSR
```sh
npm run build:ssr
```

### Build CSR
```sh
npm run build:csr
```

### Run SSR (prod)
```sh
npm run serve:ssr
```

### Run CSR (preview)
```sh
npm run serve:csr
```
